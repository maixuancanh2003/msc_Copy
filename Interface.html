User Guide

Filename Rules:

  • Do not use spaces in filenames.
  • Do not start filenames with a number.
  • Do not use accented file names, if there are special characters they will be replaced with "_".
  • Files are saved to the path: /ESP.

Text Input:

The "Enter your data here..." textbox is used for testing data transmission. Data will be sent to the ESP32 and saved.

USB Toggle:

  • If enabled, the green light will be on (for ESP32S3).
  • If disabled, the red light will be on (for PC).
